# AI PickKit

AI 선택을 쉽게. 초보자도 AI 모델과 API 비용을 비교하고 자신의 사용량에 맞는 도구를 고를 수 있는 의사결정 도구 모음입니다.

## MVP

- 초보자용 AI API 비용 계산기
- 입력·출력 토큰과 호출 수 기반 상세 계산
- AI 모델 가격·특성 비교
- 사용 목적 기반 모델 선택 도우미

## 개발

```bash
npm ci
npm run check
npm run build
git diff --check
```

자동화 작업자는 작업 전에 반드시 `AUTOMATION.md`와 `TASK_QUEUE.md`를 읽어야 합니다.
