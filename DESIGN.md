# AI PickKit Design System

Mintlify 계열의 명료한 도구·문서 경험을 AI PickKit 브랜드에 맞게 적용한다.

## 방향

- 신뢰할 수 있는 계산 도구와 읽기 쉬운 설명 문서를 한 화면 체계로 결합한다.
- 흰색 기반, 민트 포인트, 짙은 텍스트를 사용한다.
- 장식보다 입력값, 결과, 근거, 업데이트 날짜의 우선순위를 높인다.
- 초보자 모드는 일상 언어를 사용하고 고급 설정은 점진적으로 공개한다.

## 토큰

- Brand: `#16a37a`
- Brand dark: `#0d7a5b`
- Ink: `#171717`
- Muted: `#667085`
- Border: `#e6e8eb`
- Surface: `#ffffff`
- Canvas: `#f7f9f8`
- Radius: 카드 16px, 입력 10px, 버튼 10px
- Font: Inter 또는 Geist, 한국어는 Pretendard 호환 system sans-serif

## UI 원칙

- CTA는 짙은 배경 또는 민트색으로 명확히 구분한다.
- 계산 결과는 월간 예상비용을 가장 크게, 산식과 출처는 바로 아래에 표시한다.
- 모바일 360px부터 사용할 수 있어야 한다.
- 키보드 focus, 명시적 label, 충분한 대비를 보장한다.
- 가격 데이터마다 공급자 공식 출처와 마지막 확인일을 표시한다.
